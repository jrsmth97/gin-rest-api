## Golang Gin REST API

### Feature

- [x] JWT
- [x] Gorm
- [x] API Documentation Swagger
- [x] Go Playground Validator

## Command


  - Install node modules

  ```sh
  $ go get . || go mod || make goinstall
  ```

  - Build application

  ```sh
  $ go build -o main || make goprod
  ```

  - Start application in development

  ```sh
  $ go run main.go | make godev
  ```
